# MegaBox AI Assistant

MegaBox is a personal AI assistant built in Python that combines a local language model with system automation.

Features:
- Voice interaction
- Memory system
- Spotify music playback
- YouTube search
- Browser automation
- Brightness and volume control
- System command execution

Technologies:
Python, Ollama (Llama3), SpeechRecognition, PyAutoGUI,pyttsx3

Run the assistant:

python main.py
